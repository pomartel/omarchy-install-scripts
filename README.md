# Steps for setting up a new machine

1. Install the SSH keys from 1password

2. Run the gist to set up yadm and the Installation Scripts repo
   ```bash
  curl -fsSL https://gist.githubusercontent.com/pomartel/9c597f21340ce153e381c51b20a78956/raw | bash
   ```

3. Run the installation script
   ```bash
   ~/Install/INSTALL.sh
   ```
